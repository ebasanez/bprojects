Very flexible, versatile and user friendly. Colors and background are easily customizable. It's simple yet awesome appearance attracts visitors.

3.1. Header

The template header consist of 3 main parts:

  • #header-info
  • #header
  • #header-panel

#header-info div contains contact info with custom icons. You can find the icons (e.g: <span class="header-icon-pin"></span>) inside the list elements.

#header div contains the #logo and the #primary-navigation.

Editing #logo div you can easily change the home link and the logo image.

#primary-navigation is the site top navigation.

Notes

You can easily remove or replace icons in header and navigation.

<div id="header-info" class="wrapper">

3.2. Navigation

Site navigation is located inside #header div. #primary-navigation has two main parts. .menu-responsive shows menu title or the current page in mobile and table view. .menu ul element contains the navigation menuitems.

You can add sub menu items to the main menu items adding an ul tag with list elements inside. Adding .current class to list elements (and its parent element if has parent) makes the menu item selected/highlighted.

Menu icons are inside menuitems links e.g. (<span class="menu-icon-home"></span>), you can remove or replace it if you wish.

Notes

You can use unlimited submenus and the site navigation works also without javascript.

The responsive menu is visible in tablet and mobile view.

3.3. Hide info boxes

Hide info boxes can be placed over header panel. These boxes become visible on mousover.

To create hide info boxes with overlay effect the html code should be edited at two places.

Notes

To make hide info boxes the more awesome don't forget to add an overlay on header panel.

Hide info boxes are hidden in mobile view.

Add overlay to header panel

You can create an overlay over header panel which will be shown when you move the mouse over a hide info box. You should add div with and id of #header-overlay inside the #header-panel div.

<!-- #header panel -->
<div id="header-panel" class="wrapper header-parallax">
    <div class="header-overlay"></div>
    <div class="header-image header-image-home container"></div>
    <div class="page-title">
        <h1>Welcome to imbus</h1>
    </div>
</div>
<!-- /#header panel -->

Hide info box container structure

The hide info boxes hava a main container div with an id of #hide-info-container. You should put this to the top of the .main-content div inside #main div. Hide info boxes are inside #hide-info-container with classnames .hide-info-box-style1, hide-info-box-style2 or hide-info-box-style3. The difference is only the color style.

3.4. Content sliders

All sliders have the same structure. Sliders can contain any HTML elements, e.g. images, videos, etc.

Notes

To make them alive you should call carouFredsel plugin as described here.

Basic structure

You can build slides using list elements inside .slides ul tag. .slides div wrapped around with a div with custom classname(s). Testimonails slider has .testimonials, logo carousel (home.html) has .brands and .carousel, post slider has .post-slider classname. You can use these classnames as jQuery selectors in script.js to call the slider plugin. If you would like to bulid a custom slider then add an other classname and use this unique name for plugin calling. As you can see below .testimonials is the custom classname here (home.html), it is an unique slider with HTML content. Post sliders and logo carousels have other unique classnames (.post-slider and .brands, .carousel).

3.6. Portfolio loop

The basic sturcture of the blog loop is the following. The container div is a div with .portfolio classname. Inside this container there is ul element with .portfolio-post classname. This is the wrapper for portfolio posts. Portfolio posts has .post classname.

Notes

You can create filtered portfolio and unfiltered portfolio. The layout can be 2, 3 or 4 columns.

Portfolio loop without filter

You can create 2, 3 or 4 column type portfolio. To achive this simply add .span6, .span4 or .span3 classnames to portfolio posts elements. If your post has overlay don't forget to add .has-overlay classname to .post element.

<div class="portfolio">

    <!-- .portfolio-posts -->                   
    <ul class="portfolio-posts row">
        <!-- .post -->              
        <li class="post span3 has-overlay">
        </li>
        <!-- /.post -->
        <!-- .post -->              
        <li class="post span3 has-overlay">
        </li>
        <!-- /.post -->        
    </ul>
    <!-- /.portfolio-posts -->         
</div> 

Filterable Portfolio loop

If you would like to create a filterable portfolio you should also add .portfolio-isotope classname to this div. You should add the .portfolio-filter div with its inner stucture. You can set filter a tag using data-filter attribute to filter tabs. Attaching the filter tag to portfolio items you should set data-filter attribute(s) to posts. The template uses Isotope plugin to make the portfolio filterable.

<div class="portfolio portfolio-isotope">

    <ul class="portfolio-posts row">

</div>

4. Contact.php

The contact.php is the working verison of contact.html. All you should do when editing this file is to change the E-mail address to yours. You can also edit the E-mail subject or the message prefix part to entirely customise the E-mail.

Notes

Contact.php uses ajax request for form validation when javascript is enabled, but also works fine when javascript not present.

//+----------------------------------------------------------------+
//| email settings
//+----------------------------------------------------------------+
$to = "yourmail@yoursite.com"; /* you email address */
$subject ="Contact form message"; /* email subject */
$message ="You received a mail via your website contact form\n\n"; /* email messege prefix */

6.2. Slider Revolution

The template uses the Slider Revolution plugin for main sliders in home pages.

Setting sliderH variable is very important. This sets the slider height.

Notes

You can find more information about the plugin & settings in the plugin documentation. (located in assets/plugins/rs-plugin folder).

The templates call the plugin in script.js file.

Call the plugin

/* -------------------------------------------------------------------------------- /
	[11] SLIDER REVOLUTION
/ -------------------------------------------------------------------------------- */	

if (jQuery().revolution) {
    var $revSlider=$body.find('.rev-slider'),
    	sliderH=570; /* set the slider height */
	
	$revSlider.find('.fullwidthabnner').css({'height':sliderH,'maxHeight':sliderH}).revolution({    
		delay:9000,
		startheight:sliderH,
		startwidth:1020,
		
		navigationType:"none",                  
		navigationArrows:"none",        
		touchenabled:"on",                      
		onHoverStop:"on",                        
		
		navOffsetHorizontal:0,
		navOffsetVertical:20,
		
		hideCaptionAtLimit:0,
		hideAllCaptionAtLilmit:0,
		hideSliderAtLimit:0,
		
		stopAtSlide:-1,
		stopAfterLoops:-1,
		
		shadow:0,
		fullWidth:"on"	
	});	
		
	/* revslider arrows */
    $revSlider.delegate('.slider-control-prev, .slider-control-next', 'click', function(e){
		var $this=$(this), 
			$parent=$this.closest('.rev-slider');	
		e.preventDefault();
		if ($this.hasClass('slider-control-prev')) {
			$revSlider.revprev();
		} else {
			$revSlider.revnext();
		};
	});
};

6.4. Customise map

This template uses jQuery goMap plugin to create google map. You can set custom loacation, pin, title and many more things with this plugin.

Notes

Don not forget to call the 2 additional js files for the map as described here.
Code and settings are located in script.js file.

Call the plugin 

/* -------------------------------------------------------------------------------- /
	[10] GOOGLE MAP - GOMAP PLUGIN
/ -------------------------------------------------------------------------------- */	

var $headerMap=$body.find("#header-map");	
if (jQuery().goMap && google && google.maps) {
	 $headerMap.each(function(index, element) {
		var map=$(this).goMap({ 
			markers : [{ 
				address : 'Leatherhead, Surrey United Kingdom 01372 818123, United Kingdom', /* change your adress here */
				title : 'Granth', /* title information */
				icon : {
					image : 'assets/images/misc/pin_red.png' /* your custom icon file */
				}	
			}], 
			scrollwheel : false,
			zoom : 13,
			maptype : 'ROADMAP'
		});
	});
	var mapResize=false;
	$(window).resize(function(e) {
		if (mapResize) { clearTimeout(mapResize); }
		mapResize = setTimeout(function() {
			if ($.goMap.getMarkers('markers').length) {
				$.goMap.map.panTo($.goMap.getMarkers('markers')[0].getPosition());
			};
		}, 100);
	});
};
